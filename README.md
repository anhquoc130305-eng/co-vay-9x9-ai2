# Go 9x9 AI Web App

Ứng dụng AI chơi cờ vây 9x9 chạy trên web bằng Python Streamlit.

## Công nghệ

- Python
- Streamlit
- Minimax
- Alpha-Beta Pruning
- Heuristic Evaluation

## Cách chạy local

Cài thư viện:

```bash
pip install -r requirements.txt
```

Chạy app:

```bash
streamlit run app.py
```

Sau đó trình duyệt sẽ tự mở game.
